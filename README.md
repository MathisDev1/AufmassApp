# AufmassApp 📐

Eine iOS-App für Maler und Fliesenleger, die den Angebotsprozess 
von Stunden auf Minuten reduziert.

## Was die App macht

Handwerker verschwenden täglich 2–4 Stunden mit manuellen Aufmaßen 
und Angeboten. AufmassApp löst das: Raum mit dem iPhone scannen, 
Leistungen auswählen, professionelles PDF-Angebot in unter 
10 Minuten direkt auf der Baustelle versenden.

## Features

- **LiDAR-Raumscan** — RoomPlan API erkennt automatisch Wände, 
  Türen und Fenster mit ±2cm Genauigkeit
- **Automatische Flächenberechnung** — Wandfläche netto, 
  Deckenfläche, Bodenfläche, Raumhöhe, Umfang
- **Angebotskalkulation** — Leistungen per Toggle auswählen, 
  Preise/m² individuell konfigurierbar, Live-Berechnung
- **PDF-Export** — Professionelles DIN-A4-Angebot mit Firmenlogo, 
  Positionstabelle, MwSt-Ausweis, direkt per WhatsApp oder 
  Mail versenden
- **Kundenverwaltung** — Projekte und Kunden lokal speichern, 
  Angebotsstatus tracken
